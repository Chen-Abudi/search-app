# ✨ How to Get Started

## Clone Project:

```bash

git clone https://github.com/Chen-Abudi/search-app.git

```

## Enter the project directory:

```bash

cd search-app

```

## Install the Dependencies:

```bash

npm install

```

or in short

```bash

npm i

```

## Launch:

```bash

npm run start

```
